Create Gmail App Password

Follow these steps to generate an app password for secure connection

  1. Go to Google Account Security
  2. Enable 2-Step Verification
  3. Scroll to "App Passwords" section
  4. Select "Mail" as the app and "Windows Computer" as the device, then click Generate
  5. Copy the 16-character app password
  6. Use this app password instead of your regular Gmail password
